# 🎯 Goal System — Time-Based Learning (Meta + Progresso)

## 📌 Objetivo

O **Goal System** é um sistema de metas com tempo definido que guia o aprendizado do usuário de forma estruturada.

Ele responde:

> **“Onde eu quero chegar e quanto falta para isso?”**

---

## 🧠 Filosofia

> Tempo cria urgência  
> Progresso cria motivação  
> Consistência cria resultado

O sistema combina:

- ⏳ Tempo (dias restantes)
- 🎯 Meta (nível alvo)
- 📈 Progresso real (baseado em estudo)

---

## 🎯 Estrutura da Meta

Cada usuário define:

- Idioma (ex: Alemão 🇩🇪)
- Nível alvo (A1, A2, etc.)
- Duração (ex: 90 dias)

---

### 📊 Exemplo

```txt
Goal: German A1
Duration: 90 days
Start: Day 1
End: Day 90
