# 🌍 Language Learning Platform (Real-Life Focus)

## 📌 Objetivo

Plataforma pessoal de aprendizado de idiomas focada em uso real e diário.

Diferente de apps tradicionais, o foco é:
- Aprendizado prático (situações reais)
- Uso diário rápido (10–20 min)
- Sistema guiado + prática livre
- Construção de um projeto técnico sólido (React + Django + IA)

---

## 🌐 Idiomas Suportados

O sistema é multi-idioma e permite combinações:

- Português (PT)
- Inglês (EN)
- Espanhol (ES)
- Alemão (DE)

Exemplos de uso:
- PT → DE
- EN → DE
- PT → EN
- ES → DE

---

## 🧠 Filosofia do Projeto

> "Ferramenta de uso diário > Projeto perfeito"

Princípios:
- Simples e rápido
- Sem distrações
- Foco em retenção
- Foco em prática ativa (não passiva)

---

## 🧱 Arquitetura

- API REST

---

## 📐 Estrutura do Produto

### 1. Home
- Streak de dias
- Progresso geral (A1 → A2 → B1…)
- Tempo de estudo
- Botão: "Começar estudo do dia"

---

### 2. Estudo Guiado (CORE)
Sistema diário estruturado.

Cada dia contém:
- Novas frases
- Revisões
- Exercícios (input + output)

Modos:
- Leitura (DE → PT)
- Produção (PT → DE)
- Futuro: áudio

---

### 3. Cenários
Aprendizado baseado em situações reais:

- Restaurante
- Mercado
- Transporte
- Universidade
- Moradia

---

### 4. Vocabulário / Revisão
Lista pessoal de:
- Palavras favoritas
- Frases difíceis
- Revisão rápida diária

---

## 🔄 Fluxo de Uso

1. Usuário abre app
2. Vai direto para Estudo Guiado
3. Completa tarefas do dia
4. Marca palavras importantes
5. Revisa à noite
