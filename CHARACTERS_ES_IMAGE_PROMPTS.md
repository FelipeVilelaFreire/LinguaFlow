# Talkly ES A1 — Image Prompts: Todos os Personagens

> Prompts otimizados para geradores de imagem (Midjourney, DALL-E 3, Flux, Leonardo.ai, Stable Diffusion).
> Todos compartilham o mesmo prefixo de estilo para garantir coerência visual entre os personagens.
> Formato portrait 3:4 (mobile game character card).

---

## Prefixo de estilo — copiar em TODOS os prompts

```
painterly digital illustration, character portrait, Spanish colonial village setting (San Cristóbal, 16th century Mexico), warm golden afternoon light, semi-realistic, oil painting texture, rich earth tones, detailed face, mobile RPG game art, cinematic atmosphere, --ar 3:4
```

---

## Qualidade / negative (adicionar ao final de cada prompt)

```
quality: highly detailed, sharp focus, professional illustration, vivid colors
negative: blurry, deformed, extra limbs, modern clothing, neon colors, anime style, western cartoon style, photorealistic photo
```

## Notas de coerência visual

- **Paleta comum:** terra (ocre, siena, terracota), verde-escuro, índigo, crème — evitar azul saturado ou verde-limão
- **Iluminação:** golden hour afternoon ou candlelight. Nunca flash/estúdio.
- **Background:** sempre cenário de San Cristóbal — nunca fundo branco ou gradiente genérico
- **Expressões:** cada personagem tem uma expressão específica nos prompts — não trocar por genérica "sorindo"
- **Estilo:** painterly/oil — não realista fotográfico, não anime, não cartoon ocidental
- Se o gerador pedir seed, usar o mesmo seed para todos os personagens do mesmo lote
